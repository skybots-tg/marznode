# Руководство пользователя: Блокировка устройств

## 🎯 Что это такое?

Система блокировки устройств позволяет ограничивать количество устройств, с которых пользователь может подключаться к VPN. Если пользователь попытается подключиться с устройства сверх установленного лимита, подключение будет заблокировано на уровне прокси-сервера.

## 📱 Как это работает?

### Для администратора

1. **Установка лимита устройств для пользователя:**
   - Перейдите в профиль пользователя
   - Установите `device_limit` (например, 3 для трех устройств)
   - Сохраните изменения

2. **Просмотр устройств пользователя:**
   - Откройте раздел "Устройства" в профиле пользователя
   - Вы увидите список всех подключенных устройств
   - Для каждого устройства показывается:
     - IP адрес
     - Название клиента (v2rayNG, Shadowrocket, и т.д.)
     - Дата первого и последнего подключения
     - Трафик
     - Статус (активно/неактивно)

3. **Блокировка устройства:**
   - Нажмите "Заблокировать" рядом с устройством
   - Устройство будет исключено из списка разрешенных
   - При следующей попытке подключения с этого устройства - соединение будет отклонено

4. **Разблокировка устройства:**
   - Нажмите "Разблокировать" рядом с устройством
   - Устройство будет добавлено обратно в список разрешенных

5. **Удаление устройства:**
   - Нажмите "Удалить" рядом с устройством
   - Устройство будет полностью удалено из системы
   - При следующем подключении оно зарегистрируется как новое устройство (если не превышен лимит)

### Для пользователя

1. **Первое подключение:**
   - Установите VPN клиент на устройство
   - Подключитесь используя ваш ключ
   - Устройство автоматически зарегистрируется в системе

2. **Подключение с нового устройства:**
   - Если у вас не превышен лимит устройств - подключение пройдет успешно
   - Если лимит превышен - подключение будет заблокировано
   - Обратитесь к администратору для:
     - Увеличения лимита
     - Или удаления старого устройства

3. **Если подключение блокируется:**
   - Проверьте, не превышен ли лимит устройств
   - Свяжитесь с администратором
   - Администратор может:
     - Увеличить ваш `device_limit`
     - Удалить старые/неиспользуемые устройства
     - Проверить список ваших устройств

## 🔢 Примеры использования

### Пример 1: Пользователь с тремя устройствами

```
Пользователь: john_doe
Device limit: 3
Зарегистрированные устройства:
  1. iPhone (iOS) - Активно
  2. Android (v2rayNG) - Активно  
  3. Windows (v2rayN) - Неактивно
```

**Сценарий 1:** Пользователь пытается подключиться с iPad
- ❌ Подключение заблокировано (лимит 3/3 уже использован)
- Решение: Удалить Windows устройство или увеличить лимит до 4

**Сценарий 2:** Администратор удаляет Windows устройство
- ✅ Теперь можно подключить iPad (2/3 устройства)

### Пример 2: Пользователь без лимита

```
Пользователь: admin_user
Device limit: не установлен (null)
```

- ✅ Может подключаться с любого количества устройств
- Устройства все равно отслеживаются для статистики

### Пример 3: Пользователь с лимитом 0

```
Пользователь: suspended_user
Device limit: 0
```

- ❌ Не может подключиться ни с одного устройства
- Используется для временной блокировки доступа

## 📊 Информация об устройстве

Для каждого устройства система собирает:

- **Remote IP:** Последний IP адрес подключения
- **Client Name:** Название клиента (v2rayNG, Shadowrocket, и т.д.)
- **User Agent:** Дополнительная информация о клиенте (если доступно)
- **TLS Fingerprint:** Уникальный отпечаток TLS соединения (если доступно)
- **First Seen:** Дата и время первого подключения
- **Last Seen:** Дата и время последнего подключения
- **Total Usage:** Общий трафик с этого устройства
- **Status:** Активно/Неактивно

## ⚙️ Настройки администратора

### Device Limit (Лимит устройств)

- **null (не установлен):** Без ограничений
- **0:** Блокировать все устройства (полная блокировка)
- **1-N:** Разрешить N устройств

### Enforcement (Применение ограничений)

- **Включено:** Блокировка происходит на уровне прокси-сервера
- **Выключено:** Устройства отслеживаются, но не блокируются

## 🔐 Безопасность

### Как определяется устройство?

Устройство идентифицируется по "fingerprint" - уникальному отпечатку, который вычисляется на основе:
- User ID
- Название клиента
- TLS fingerprint
- User Agent

Это означает:
- ✅ Замена SIM-карты НЕ создает новое устройство
- ✅ Смена IP адреса НЕ создает новое устройство
- ⚠️ Переустановка VPN клиента МОЖЕТ создать новое устройство (зависит от клиента)
- ⚠️ Использование разных VPN клиентов создает разные "устройства"

### Конфиденциальность

- IP адреса хранятся только для последнего подключения
- TLS fingerprints и User Agents собираются только если доступны
- История подключений хранится локально на сервере
- Данные не передаются третьим лицам

## 📝 FAQ

### Q: Почему мое устройство заблокировано?

**A:** Возможные причины:
1. Превышен лимит устройств (проверьте `device_limit`)
2. Устройство заблокировано администратором
3. Устройство удалено из списка разрешенных

**Решение:** Свяжитесь с администратором.

### Q: Сколько устройств я могу использовать?

**A:** Это зависит от вашего `device_limit`:
- Если не установлен - без ограничений
- Если установлен - не более указанного количества

### Q: Как посмотреть свои устройства?

**A:** Обратитесь к администратору. Он может:
- Показать список ваших устройств
- Удалить старые устройства
- Изменить лимит

### Q: Что происходит при переустановке VPN клиента?

**A:** Зависит от клиента:
- Некоторые клиенты сохраняют fingerprint - устройство останется тем же
- Другие генерируют новый fingerprint - создастся новое "устройство"
- Если превышен лимит - нужно удалить старое устройство

### Q: Можно ли использовать VPN на телефоне и компьютере?

**A:** Да, если ваш `device_limit` ≥ 2:
- Каждое физическое устройство считается отдельно
- Убедитесь, что лимит позволяет использовать оба устройства

### Q: Что означает "Неактивное" устройство?

**A:** Устройство считается неактивным, если:
- Не было подключений более 5 минут (по умолчанию)
- Неактивные устройства все еще занимают место в лимите
- Администратор может удалить неактивные устройства

### Q: Как увеличить лимит устройств?

**A:** Только администратор может изменить `device_limit`:
- Обратитесь к администратору с просьбой
- Объясните, почему вам нужно больше устройств

## 🛠️ Troubleshooting

### Проблема: Не могу подключиться

**Проверьте:**
1. Правильность ключа/конфигурации
2. Не превышен ли лимит устройств
3. Не заблокировано ли ваше устройство администратором
4. Активна ли ваша подписка

**Решение:** Свяжитесь с администратором с описанием проблемы.

### Проблема: "Слишком много устройств"

**Причина:** Превышен `device_limit`

**Решение:**
1. Попросите администратора показать список ваших устройств
2. Удалите неиспользуемые устройства
3. Или попросите увеличить лимит

### Проблема: Каждое подключение создает новое устройство

**Причина:** Возможно, VPN клиент не сохраняет fingerprint

**Решение:**
1. Используйте другой VPN клиент
2. Или попросите администратора увеличить лимит
3. Не переустанавливайте клиент без необходимости

## 📞 Поддержка

При возникновении проблем обращайтесь к администратору со следующей информацией:

- Ваше имя пользователя
- Название устройства/VPN клиента
- Описание проблемы
- Когда проблема началась
- Скриншот ошибки (если есть)
